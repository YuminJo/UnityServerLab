# C# / .NET Server & Unity Study

## English

This repository contains my study notes and example code related to **C#, Unity, and .NET server development**.

The purpose of this project is to better understand the technologies used in game development and server programming.  
It includes small experiments, practice code, and concept demonstrations.

Topics covered in this repository may include:

- C# programming fundamentals
- Multithreading and concurrency
- Memory management
- .NET server architecture
- Networking basics
- Asynchronous programming (`async/await`, `Task`)
- Thread synchronization (`lock`, `Monitor`, `Interlocked`, `volatile`)
- Unity scripting and C# patterns
- Performance considerations

Most examples are simple experimental implementations used to understand how these systems work internally.

This repository serves as a personal learning archive while studying **game development and server engineering**.
